# Ehtisham Aflak | Modern Personal Portfolio

Welcome to my personal portfolio! This project is a sleek, responsive, and interactive showcase of my journey as a **Software Engineering student** and **Front End Developer**.

---

## Features

- **Dynamic Typing Animation:** Custom typewriter effect showcasing my roles as a Frontend Developer and UI Designer.  
- **Glassmorphism UI:** Modern frosted-glass design with smooth CSS transitions and neon glow effects.  
- **Interactive Sidebar:** Smooth navigation with active-state tracking and hover-glow animations.  
- **Responsive Design:** Fully optimized layout for both desktop and mobile devices.  
- **Project Gallery:** Grid-based display of my latest work with hover-up animations.
